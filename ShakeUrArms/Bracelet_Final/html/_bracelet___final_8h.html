void getMessage ( char  buffer[],
uint8_t  maxSize,
char  type

Récupère le buffer du périphérique bluetooth ou standard et l'affiche sur le périphérique bluetooth ou standard.

On regarde si mySerial et Serial sont vide et compte avec delay le temps pour parcourir le message envoyé on affiche donc le message sur le périphérique bluetooth ou standard durant ce même delay

Paramètres
bufferTableau de char qui contient le message envoyé.
typeb pour la lecture sur le périphérique bluetooth et s pour la sortie standard (bracelet).
